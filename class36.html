If-else statements

The Javascript if-else statement is used to execute the code wether the condition is true or false

there are 2 types of if statements in Javascript

1. if statement

2.if-else statement

IF evaluates the content only if expression/condition is true. The signature of JavaScript if statement is given below. If(condition){ .. code to be executed when it is true .. }

IF-ELSE evaluates the content whether condition is true of false. The syntax of JavaScript if-else statement is given below. if(expression){ //content to be evaluated if condition is true } else{ //content to be evaluated if condition is false }
